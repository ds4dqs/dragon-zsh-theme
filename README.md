# dragon-zsh-theme

dragon is a minimalistic zsh prompt theme for git users

## Feature

*   Pretty prefix prompt signature
*   Pretty color
*   Current directory
*   Time stamp (in ISO 8601 format)
*   D-Day counter (require Node.JS)
*   SSH session indicator
*   Git prompt info
*   Git status info
